# ICare
ICare is an app I build to remind (or force ^_^) me to take a break from the screen every 20min becosue my doctor said so. I figured this would help other people, so I just started building it. It has a nice UI, it is VERY lightwayght, and it does its job.

# Instalation
Head over to https://github.com/SilentSword123456/ICare/releases and chose the latest release. Download it, and then run it (it dosent need Admin privileges). You should be able to acces sthe dashboard via a systemtray icon by double clicking it, and quit the app by right clicking on the icon and pressing quit.
